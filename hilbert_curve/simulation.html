Rendering Hilbert Curves







Brief Description

The Hilbert curve is a space-filling curve. The ideal Hilbert curve is achieved as the order tends to infinity. All other curves are pseudo Hilbert curves. "Space-filling curve" is one in which covers an entire 2D space. There is also a mapping between the real numbers in [0, 1] and points on the 2D space.
It is commonly used in games and rendering software.



Note:
  1. Although most implementation recursivly draw the curve, the above implementation uses an iterative method.
  2. Hilbert curves of upto 10th order can be comfortably drawn on lower-end devices too.
